# JavaScript 的运算符
## 算术运算符
 - 给定y = 1，下面的表格解释了这些算术运算符：

运算符 | 描述 | 案例 | X的结果 | y的结果
---|---|---|---|---
+ | 加法 | x = y + 1 | 2 | 1
- | 减法 | x = y - 1 | 0 | 1
* | 乘法 | x = y * 2 | 2 | 1
/ | 除法 | x = y / 2 | 0.5 | 1
% | 取模(取余)| x = y % 2 | 1 | 1
++| 自增 | x = ++y | 2 | 2
|||x = y++ | 1 | 2
--| 自减 | x = --y | 0 | 0
|||x = y-- | 1 | 0

## 赋值运算符
 - 给定 x=10 和 y=5，下面的表格解释了赋值运算符:

运算符 | 例子 | 等同于 | 运算结果
---|---|---|---
= | x = y |  | x = 5
+= | x += y | x = x + y | x = 15
-= | x -= y | x = x - y | x = 5
*= | x *= y | x = x * y | x = 5
/= | x /= y | x = x / y | x= 2
%= | x %= y | x = x % y | x = 0

## JavaScript 比较运算符
运算符 | 描述
---|---
==	| 等于
===	| 等值等型
!=	| 不相等
!==	| 不等值或不等型
>	| 大于
<	| 小于
>=	| 大于或等于
<=	| 小于或等于
?	| 三元运算符
